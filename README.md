# PolyMinder

PolyMinder is a comprehensive project that includes both backend and frontend components. This README provides an overview of the project structure and guides you to the respective folders for more detailed information.

## Project Structure

- `backend/`: Contains the backend code and related resources.
- `web_interface/`: Contains the frontend code and related resources.

## Getting Started

### Backend

For detailed information on setting up and running the backend, please refer to the [backend README](backend/README.md).

### Frontend

For detailed information on setting up and running the frontend, please refer to the [web_interface README](web_interface/README.md).
